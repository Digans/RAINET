<h3><a id="user-content-pipeline-description" class="anchor" aria-hidden="true" href="#pipeline-description"><svg class="octicon octicon-link" viewBox="0 0 16 16" version="1.1" width="16" height="16" aria-hidden="true"><path fill-rule="evenodd" d="M4 9h1v1H4c-1.5 0-3-1.69-3-3.5S2.55 3 4 3h4c1.45 0 3 1.69 3 3.5 0 1.41-.91 2.72-2 3.25V8.59c.58-.45 1-1.27 1-2.09C10 5.22 8.98 4 8 4H4c-.98 0-2 1.22-2 2.5S3 9 4 9zm9-3h-1v1h1c1 0 2 1.22 2 2.5S13.98 12 13 12H9c-.98 0-2-1.22-2-2.5 0-.83.42-1.64 1-2.09V6.25c-1.09.53-2 1.84-2 3.25C6 11.31 7.55 13 9 13h4c1.45 0 3-1.69 3-3.5S14.5 6 13 6z"></path></svg></a>Pipeline description:</h3>
<ul>
<li>RAINET DB preparation:
<ul>
<li>ReadCatrapid.py (process catRAPID software data (licenced))</li>
<li>InsertionStrategy.py (for creating a RAINET database)</li>
<li>AnalysisStrategy.py (for filtering / stats of RAINET database)</li>
</ul>
</li>
<li>Enrichment analysis
<ul>
<li>EnrichmentAnalysisStategy.py (produces main RNA-complex enrichment results)</li>
<li>FilterEnrichmentResults.py (for filtering / stats Enrichment analysis results)</li>
<li>CommonLncRNAProteinDisease.py (for matching lncRNA and protein disease)</li>
</ul>
</li>
<li>Other post-analysis:
<ul>
<li>PrioritizeCandidates.py (for selecting enrichments with known interactions)</li>
<li>LncRNAGroupOddsRatio.py (for evaluating overlap of groups of lncRNAs against functional lncRNAs)</li>
</ul>
</li>
</ul>
